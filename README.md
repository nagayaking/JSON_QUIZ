# JSON QUIZ

`{質問: 答え}` 形式のJSONファイルを読み込んで遊べる、1ファイル完結のクイズアプリです。

## 使い方

1. `quiz.html` をブラウザで開く
2. 「ファイルを選ぶ」からJSONファイルを選択する
3. ランダムな順序で1問ずつ出題されるので、回答欄に答えを入力して Enter(または「採点する」)
4. 正誤判定後、Enter(または「次の設問へ」)で次の問題へ
5. 全問終わったら成績が表示され、次のいずれかで再挑戦できる
   - 間違えた問題だけ
   - 正解した問題だけ
   - 全部の問題
   (0問の選択肢は自動的に押せなくなります)

右上の「別のファイルを読み込む」で、いつでも別のJSONに切り替えられます。

## JSONファイルの形式

```json
{
  "質問1": "答え1",
  "質問2": "答え2"
}
```

サンプル: `sample_quiz.json`

## 採点ルール

- 前後の空白を除いて**完全一致**した場合のみ正解になります(大文字・小文字や表記ゆれは区別されるので注意)

## ファイル構成

- `quiz.html` … アプリ本体(これ1つだけで動作します。インターネット接続不要)
- `sample_quiz.json` … 動作確認用のサンプル問題
